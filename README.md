# Portfolio

A personal portfolio website showcasing certifications and professional achievements.

## Features

- **Responsive Design**: Mobile-friendly layout that adapts to all screen sizes
- **Certifications Section**: Displays earned certifications with download links
- **Modern Styling**: Clean and professional design with hover effects

## Project Structure

```
Portfolio/
├── Mica.html          # Main HTML file
├── Mica.css           # Styling for the portfolio
└── certificates/      # Directory containing certificate PDFs and badges

```

## Certifications

The portfolio includes the following certifications:

- **ISO/IEC 27001:2022 Lead Auditor** (Mastermind Assurance — 2025)
- **Introduction to Cybersecurity** (Cisco — 2025)
- **ISO/IEC 42001:2023 Lead Auditor** (Mastermind Assurance — 2025)

Each certification can be downloaded as a PDF by clicking the download link on the certificate card.

## Styling

The portfolio uses a responsive CSS grid layout with:

- Gradient background for the certifications section
- Card-based design for each certification
- Hover effects for better interactivity
- Mobile-first responsive design
